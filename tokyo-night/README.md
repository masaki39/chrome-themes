# Tokyo Night Theme for Chrome

A clean, dark Chrome theme inspired by the popular Tokyo Night color palette. This theme provides a comfortable, low-contrast dark experience with carefully chosen accent colors.

## Color Palette

The Tokyo Night theme uses the following colors:

| Color Name | Hex | RGB | Usage |
|------------|-----|-----|-------|
| Background | `#1a1b26` | `26, 27, 38` | Frame, NTP background |
| Background Alt | `#16161e` | `22, 22, 30` | Toolbar, tabs, omnibox, inactive tabs |
| Red | `#f7768e` | `247, 118, 142` | Active tab text |
| Green | `#9ece6a` | `158, 206, 106` | NTP section highlights |
| Yellow | `#e0af68` | `224, 175, 104` | Toolbar button icons |
| Blue | `#7aa2f7` | `122, 162, 247` | Links, bookmarks |
| Magenta | `#bb9af7` | `187, 154, 247` | Buttons, incognito frame, headers, section links |
| Cyan | `#7dcfff` | `125, 207, 255` | NTP section text |
| Gray | `#a9b1d6` | `169, 177, 214` | General text, active tab background text, omnibox text |
